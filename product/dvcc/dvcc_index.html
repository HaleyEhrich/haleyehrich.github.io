1 简介

注：该部分内容大约 120 字，阅读该部分大约需要 1 分钟

DVCC simulation environment(以下简称DVCC)是一款适用于计算机组成原理课程的模拟软件。使用QT C++构建。相较于其它版本，DVCC可以脱离硬件设备，独立运行。可以选择读取微代码文件自动运行，也可选择手动配置信号运行。

2.1.1 界面概述

2.1.2 菜单栏介绍

Fig-2-2 菜单栏

菜单栏共有5项，分别为文件、运行、工具、选项、关于菜单栏。

  1. 文件菜单栏：
    在该菜单栏你可以
    • 选择新建一个窗口
    • 打开、保存、另存微代码文件

      微代码文件格式为 .dvc/ .txt
      保存可以将目前RAM、ROM内内容保存到打开的微代码文件（保存文件格式为.dvc，注：非2位地址标准的微代码，而是DVCC专用4地址微代码）

    • 黏贴微代码
    • 打开内存查看器

      (内存查看器.详细信息见 2.1.7.1 RAM面板 && 2.1.7.2 ROM面板 )

  2. 运行菜单栏：
    在运行菜单栏中你可以
    • 选择打开关闭微代码信息框

      (微代码信息框见 2.1.4微代码信息框 )

    • 配置运行设置(在版本 1.4.2.2 该菜单项仅能配置指令运行的间隔时间)
    • 打开运行日志浮动面板
    • 保存运行日志
  3. 工具菜单栏：
    • RAM/ROM查看器面板
    • 部件选择

      选择加载的部件，未加载的部件在运行过程中所有的操作将被忽略。

    • 微指令转换器

      实现DVCC 4 地址微代码转为标准 2 位地址微代码
      (转换器使用说明见 2.1.11.2 DVCC转换器 )

    • 其它面板
      注:目前未有任何功能
  4. 选项菜单栏：
    • 恢复默认窗口大小

      将应用窗口恢复为设置的大小 (推荐大小为1500x900)
      默认窗口大小设置路径为: 选项菜单栏->设置->通用->默认窗口大小

    • 字体设置
      对目前运行的窗口设置字体 (注：该字体更改不会影响默认设置)
      注：请注意，选择的自定字体注意侵权问题，推荐使用"Harmony OS Sans SC"作为字体
    • 主题设置
      对目前运行的窗口设置主题 (注：该主题更改不会影响默认设置)
    • 设置
      详细信息见
  5. 关于菜单栏

2.1.3 电路图区域介绍

Fig-2-3 电路图

在电路图区域,各部件的数据位于其中心,需要注意,注入ALU等单元仅在输出结果或输入数据时才会显示数据。

各个信号的标注为该信号是否有效，例如LDDR1: 1表示为LDDR1信号有效。

特例：CN在有效时表示无进位。

2.1.4 微指令信息区域

Fig-2-4 微指令信息框

微指令信息框内有目前正在执行的微代码以及接下来一条微代码。

在微代码信息框内列的含义依次为：微指令序号、微指令6位16进制内容、M信号、CN信号、WE信号、B1B0信号、A字段内容、B字段内容、C字段内容、U5-U0内容、输出数据到总线的部件、从总线输入数据的部件、跳转方式。

-注-：U5-U0为微代码所填入的转移地址(序号)，不一定为下一条微指令地址(序号)。
